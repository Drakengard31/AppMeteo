# Météo App 🌤

Application météo React avec API OpenWeatherMap — météo en temps réel + prévisions 5 jours.

## Stack
React 18 · Vite · API REST (OpenWeatherMap)

## Installation
```bash
npm install
```
Remplace `REMPLACE_PAR_TA_CLE_API` dans `src/App.jsx` par ta clé OpenWeatherMap (gratuite sur openweathermap.org).
```bash
npm run dev
```
